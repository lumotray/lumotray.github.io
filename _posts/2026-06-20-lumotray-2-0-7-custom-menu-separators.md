LumoTray 2.0.7 is out. This one is a small but quality-of-life update with a frequently requested menu feature and a few wallpaper fixes.

## What's new

**Custom menu separators.** You can now add visual separators between menu entries in the custom menu builder. If you have a long menu with multiple sections -- launchers at the top, scripts in the middle, folders near the bottom -- separators make it readable at a glance. Just add a separator entry in the editor where you want the break.

## What's fixed

**Live wallpapers on unusual display setups.** If you use a display topology with negative coordinates (for example, a secondary monitor positioned above or to the left of the primary in Windows display settings), live wallpapers like video, webpage, or Matrix mode were not applying correctly. That should work now.

**Edge case image saves.** A handful of scenarios where wallpaper images would fail to save properly have been cleaned up.

## Get it

The update is live on the Microsoft Store and should install automatically if you have automatic updates turned on. Otherwise, grab it from the [download page](https://lumotray.com/download/blog_post_2_0_7).
